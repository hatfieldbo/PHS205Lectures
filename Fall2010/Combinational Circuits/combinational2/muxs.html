Muliplexers

Basic Features     

  • n select lines
  • 2n data inputs
  • 1 to 2 data output

  • The size of any multiplexers is measured by   2n×1

  • Example: An 8×1 MUX

    • 2n = 8 data inputs
    • n = 3 select lines
    • inputs selected by n control inputs 

    • 1 data output ( 1 data output with any size MUX )

    • 1-2 enable lines

  • How about:    a 16×1 MUX,     a 4×1 MUX
