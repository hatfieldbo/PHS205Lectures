Examples of Boolean Algebraic Manipulations

Simplification/Minimization

  • F (A, B, C, D) = (A + B)' + (CD'A)'
  • G (a, b, c, d, e) = (a + ab)' + ( cde)'
  • f (A, B, C, D) = (AD + A'C)(B'(C + BD'))

Proof of Boolean Equations

  • XY + YZ + XZ' = YZ + XZ'
  • (A + B')B = AB
  • (a + b)(a' + c) = ac + a'b
  • ABC + A'BC + A'B'C + A'BC' + A'B'C' = BC + A'B' + A'C'

Convert to canonical SOP

  • F (A, B, C, D) = (A + B)' + (CD'E)'
  • F(W, X, Y, Z) = (W +X' + Y')(W' + Z')(W + Y)

Find the complement function and express in SOP form

  • F (A, B, C, D) = (A + B)' + (CD'A)'
  • G (a, b, c, d) = a(b + cd)
  • H (S, T, U, V, W, X, Y, Z) = X(Y + ZW' + V'S)
  • f (A, B, C, D) = (A + (BCD)')((AD)' + B(C' + A))
