(a) derive a Boolean expression for F2 in Canonical SOP [i.e., standard SOP or Sum of Minterms] form,

F2(A,B,C) = m3 + m5 + m6 + m7 = S (3, 5, 6, 7) = A'BC+AB'C+ABC'+ABC

(b) simplify this Canonical SOP to its minimal SOP expression,

F2(A,B,C) = A'BC+AB'C+ABC'+ABC = AB + BC + AC

Note: See your class notes for details of the simplification.

(c) convert this minimal SOP expression to its minimal POS expression

Equation F2(A,B,C) Boolean Laws/Rules applied
F2(A,B,C) = AB + BC + AC = B(A+C) + AC distributive law: xy + xz = x(y + z)
= (B + AC)((A+C) + AC) distributive law: x + yz = (x + y)(x + z)
= (B+A)(B+C)(A+ C + AC) distributive law: x + yz = (x + y)(x + z)
= (A+B)(B+C)(A+C) absorption law: x + xy = x

Note: To convert from a minimal SOP to a minimal POS, we need to apply distributive laws of Boolean algebra repetitively.

(d) convert function F2(A,B,C) to an NAND-NAND expression that will result to an NAND-NAND circuit

Convert function F2(A,B,C) to an NAND-NAND expression/circuit Boolean Laws/Rules applied
F'(A, B, C) = (AB + BC + AC)'' applied rule: x'' = x
= ((AB)' (BC')' (AC)')' DeMorgan's Theorem: (x + y)' = x'y'
  and (x + y + z)' = x'y'z'

(e) convert function F2(A,B,C) to an NOR-NOR expression that will result to an NOR-NOR circuit, and

Convert function F2(A,B,C) to an NAND-NAND expression/circuit Boolean Laws/Rules applied
F(A, B, C) = (A+B)(B+C)(A+C)'' applied rule: x'' = x
= ((A+B)' + (B+C)' + (A+C)')' DeMorgan's Theorem: (xy)' = x' + y'
  and (xyz)' = x' + y' + z'

(f) find the inverse (or the complement) function of F2(A,B,C) and give your answer in SOP form.

Convert function F2(A,B,C) to an NAND-NAND expression/circuit Boolean Laws/Rules applied
F'(A, B, C) = (AB + BC + AC)' DeMorgan's Theorem: (x+y)' = x' y'
= (AB)'(BC)'(AC)' and (x+y+z)' = x'y'z'
= (A' + B')(B' + C')(A' + C') DeMorgan's Theorem: (xy)' = x' + y'
= (A'B' + A'C' + B'B' + B'C')(A' + C') Distributive law: x(y + z) = xy + xz
= (A'B' + A'C' +B' +B'C')(A' + C') Boolean rule: xx = x
= (A'C' + B')(A' + C') Absorption rule: x + xy = x
= A'C'A' + A'C'C' + A'B' + B'C' Distributive law
= A'C' + A'C' + A'B' + B'C' applied: xx = x
= A'C' + A'B' + B'C' applied: xx = x
