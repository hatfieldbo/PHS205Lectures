Base Conversion Algorithms

  • Base-r => Base-10: Apply the expansion formula

 

  • Base-10 => Base-r:  Apply the following algorithms
    1. convert integer and fraction separately
    2. Integer portion

      Repetitively divide the decimal value by r

      Each remainder is another digit of the base-r number (RIGHT to LEFT)

    3. Fraction portion

            Repetitively multiple the decimal value by r

            Each integer is another digit of the base-r number (LEFT to RIGHT  )
