Memory Made of Flip-Flops [Static RAM]

  • Number of Address Bits <=> Number of Address Lines
  • Number of Bits / Memory Word <=> Number of Data Lines
  • An Example
    • a very small memory made from flip-flops
    • one flip-flop per bit
    • 4 memory word with 3 bits per word ( 4 X 3)
    • 2 address lines and 3 data input lines and 3 data output lines

 

 

 

  • Address lines: A1A0

  • Data input lines: I2I1I0 (writing)

  • Data output lines: O2O1O0 (reading)

  • CS - Chip Select ( CS = 1 to select, CS = 0 to disable the memory chip)

  • RD - ReaD ( RD = 1 to read, RD = 0 to write)

  • OE - Output Enable (OE = 1 to allow output, OE = 0 output all 0)

  • Tri-state gates (high, low, high-impedance)

    • noninvertable buffer

                high-impedance:  control = 0
